# CVBuilder Pro

Build a professional resume in minutes. No login or installation needed. Works in any browser.

## Features
- Step-by-step CV builder with live preview
- 3 professional templates (Classic, Modern, Minimal)
- AI resume score and improvement tips
- ATS checklist
- Download as PDF
